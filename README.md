# Chad Morton — Luxury Real Estate Expert

This is the official website for Chad Morton, a premier real estate agent specializing in the Washington DC and Maryland luxury markets.

## Features
- **Responsive Design**: Optimized for all devices.
- **Modern Aesthetics**: Premium, high-contrast design using Navy and Gold accents.
- **TV & Media Gallery**: Showcasing features from *The American Dream TV* and more.
- **Interactive Contact Form**: Integrated with FormSubmit for direct inquiries.
- **Performance Optimized**: Built with React and Vite for lightning-fast load times.

## Tech Stack
- **Framework**: React
- **Build Tool**: Vite
- **Styling**: Vanilla CSS with modern best practices
- **Animations**: Framer Motion

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/kirillrealtor/ChadMortonwebsite.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production
To create a production-ready bundle:
```bash
npm run build
```
